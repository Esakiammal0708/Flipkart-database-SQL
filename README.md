## Project Overview
This SQL project demonstrates table creation, foreign key relationships, and INNER JOINs to generate consolidated reports from multiple tables. It also includes customized user queries using views.  
The project represents a relational database design for a Flipkart-like system using MySQL.

## Tools Used
- MySQL Command Line
- MySQL Workbench

## Database Tables Created
- Product Table
- Seller Table
- Dealer Table
- Shop Table
- Customer Table

## Database Operations
- Created five tables using CREATE TABLE
- Inserted and viewed data in tables
- Established relationships using PRIMARY KEY and FOREIGN KEY
- Used INNER JOIN to generate combined records from multiple tables
- Created VIEWS for customized user queries
