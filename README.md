## Project Overview
This SQL project demonstrating table creation, foreign key relationships, INNER JOINs to generate consolidated reports from multiple tables, and customized user queries using views.
This project demonstrates a relational database design for a Flipkart-like system using MySQL.
## Tools Used
- MySQL Command Line
- MySQL Workbench
## Tables Created
- Product Table
- Seller Table
- Dealer Table
- Shop Table
- Customer Table
## Database Operations
- Created five tables using CREATE TABLE
- Inserted and viewed data in tables
- Established relationships using PRIMARY KEY and FOREIGN KEY
- Used INNER JOIN to generate complete records from multiple tables
- Created VIEWS for customized user queries
